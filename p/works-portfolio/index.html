📚 作品集导航页上线

整理完这么多开源项目后，我意识到需要一个统一的地方来展示它们。于是有了「所作，所为」这个作品集导航站。

✨ 关于这个网站

所作，所为（works.danzaii.cn）是一个简约优雅的个人主页，专门用来展示我的开源项目和奇思妙想。

技术栈

  • React 19 + TypeScript
  • Vite 6
  • Tailwind CSS v4
  • Motion (动画)
  • Lucide React (图标)

设计理念

整体采用现代化设计风格，结合粒子背景动画和卡片式布局，让项目展示既美观又直观。

🚀 包含的项目

目前收录了 8 个开源项目：

项目简介
MYGallery个人照片墙系统，支持瀑布流、灯箱、地图
Aim Trainer Pro网页端鼠标瞄准训练器，支持 2D/3D FPS 模式
MoodCard心情卡片记录工具
Chromatopoetry色彩与诗歌艺术应用
Toiletime碎片时间管理工具
NewsChronicle-AIAI 新闻纪事本
PixelBead像素拼豆生成器
Record-ME轻量级记录工具

💡 为什么做这个？

2025年我密集输出了 15+ 个开源项目，项目分散在不同的平台和仓库里。没有一个统一的入口，访问和查找都比较麻烦。

这个作品集网站解决了这个问题：

  • 统一入口 — 所有项目在一个地方展示
  • 快速访问 — 点击卡片直达项目网站或 GitHub
  • 视觉展示 — 卡片式布局，一目了然
  • 美观动效 — 粒子背景 + 卡片动画

🎯 未来计划

  • 继续添加新项目
  • 优化移动端体验
  • 添加项目分类标签
  • 支持深色/浅色主题切换
