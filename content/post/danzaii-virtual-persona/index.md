最近我想认真整理一下 **danzaii.cn** 这个域名背后到底是什么。

它好像已经不只是一个博客入口了。博客、作品集、照片墙、AI 生图工作室、拼豆工具、配色工具、占卜应用、瞄准训练器……这些小站散落在不同的子域名里，但它们其实都指向同一个东西：一个属于我的、不断长出来的个人数字宇宙。

所以这次我试着给 danzaii.cn 做一个虚拟形象。不是那种很商业的品牌吉祥物，而是一个能代表这个网站气质、也能代表站点所有者创作习惯的角色。

她/他叫：**丹祈 DanQi**。

## danzaii.cn 是什么？

如果只看域名，danzaii.cn 像是一个个人主页。但实际打开之后，会发现它更像一张地图。

### 1. 博客：一只小羊羔的窝

主站会跳到 [blog.danzaii.cn](https://blog.danzaii.cn)，标题是「**一只小羊羔的窝**」。这里是最核心的记录区：技术笔记、生活随笔、项目介绍、AI 绘图、旧博客迁移，还有一些来自爱莉希雅的问候。

博客的气质很明确：不是冷冰冰的技术档案，而是一个带有二次元审美、个人情绪和生活痕迹的小窝。它既会写部署、项目、开源，也会写若叶睦、爱莉希雅、照片和一些柔软的片段。

### 2. 作品集：所作，所为

[works.danzaii.cn](https://works.danzaii.cn) 是项目导航页，名字叫「**所作，所为**」。这个名字我很喜欢，因为它不是简单地说「作品集」，而是在说：这些东西都是我做过的、试过的、折腾过的。

里面收着很多项目：MYGallery、AIMBOT、MoodCard、Chromatopoetry、Toiletime、NewsChronicle-AI、PixelBead、Record-ME、Tianji……有些偏工具，有些偏审美，有些偏游戏，有些偏 AI。它们共同组成了 danzaii.cn 的「行动力」部分。

### 3. 照片墙：所见，所闻

[photo.danzaii.cn](https://photo.danzaii.cn) 叫「**所见，所闻**」。这个名字和「所作，所为」刚好形成一组：一个放做出来的东西，一个放看见过的东西。

照片墙让整个站点不只是技术展示，也有真实生活的触感。相机、旅途、日常、光线、街景，这些东西会把一个开发者从代码里拉回现实世界。

### 4. AI 生图工作室：LUMINA Atelier

[aipic.danzaii.cn](https://aipic.danzaii.cn) 是 AI-Image-Studio，也就是现在的 **LUMINA Atelier**。它负责把 prompt、参考图、工作区和灵感画廊串起来，用 GPT Image-2 生成图像。

之前博客里的若叶睦图，还有这篇文章里的虚拟形象图，都是通过这个站点生成的。它很像 danzaii.cn 的「想象力接口」：把脑子里还没有成形的画面，先温柔地接住。

### 5. 其他小站：拼豆、配色、占卜、训练器

还有一些更具体的小项目：

| 子站 / 项目 | 气质 |
|---|---|
| PixelBead | 像素、拼豆、手作、AI 生成 |
| Chromatopoetry | 色彩、诗意、灵感提取 |
| Tianji | 星象、塔罗、周易、神秘学 |
| AIMBOT | 游戏、性能、交互训练 |
| Inside-ME | 聊天记录、自我蒸馏、和过去的自己对话 |

所以如果要总结 danzaii.cn，我会这样说：

> **它是一个温柔但高产的个人数字宇宙：用博客保存记忆，用代码搭建工具，用照片记录生活，用 AI 延展想象，用二次元审美给这一切包上一层柔软的光。**

## 站点所有者的关键词

从这些内容里，可以提炼出一些很稳定的关键词：

- **技术创作者**：React、TypeScript、Go、Python、Vercel、Docker、GitHub Actions，都在项目里出现过。
- **生活记录者**：从 2021 年开始写博客，持续记录成长、项目和日常。
- **二次元爱好者**：爱莉希雅、若叶睦、崩坏3、明日方舟、崩铁、绝区零等都出现在站点气质里。
- **AI 工具搭建者**：不是只用 AI，而是会把 AI 做成自己的工作流工具。
- **摄影与猫猫爱好者**：照片墙、相机、三只小猫，是很适合视觉化的个人符号。
- **柔软但能折腾**：整体语气很温柔，但项目产出又很密集。

这几个关键词组合起来，就不是一个传统「程序员头像」能概括的。它需要一点小羊羔的柔软，一点开源开发者的执行力，一点 AI 工坊主的魔法感，一点摄影记录者的现实感，还有一点二次元角色设定的精致度。

## 虚拟形象：丹祈 DanQi

### 基础设定

| 项目 | 设定 |
|---|---|
| 名字 | 丹祈 DanQi |
| 别称 | 小羊工坊主 / Lumina Shepherd |
| 定位 | danzaii.cn 的站点虚拟形象与创作引导者 |
| 年龄感 | 18-22 岁的青年感 |
| 性别表达 | 中性偏少年感，可男可女，不做强性别化设定 |
| 头发 | 中长发，银白到浅粉渐变，发尾微卷 |
| 眼睛 | 樱粉与湖蓝渐变 / 异色感 |
| 代表色 | 羊绒白、樱花粉、夜空紫、湖光蓝、终端黑、暖金 |
| 代表物 | 小羊角发饰、相机、发光画笔、像素拼豆、三只猫、樱花、终端窗口 |

丹祈不是一个完全脱离现实的角色。她/他更像 danzaii.cn 的拟人化：会写代码，会修部署，会拍照片，会做 AI 生图，也会把很多没用但可爱的东西放进自己的小窝。

### 性格设定

丹祈的性格关键词是：**温柔、内向、行动力强、审美驱动、技术浪漫主义**。

平时看起来有点安静，说话不会很锋利，但一旦有了想法，就会开始搭项目、调接口、写博客、做图、部署上线。她/他相信技术不只是效率工具，也可以是保存记忆、表达情绪、制造美感的方式。

她/他的口头禅可能是：

> 「先做出来看看吧。」  
> 「这张图的光还可以再温柔一点。」  
> 「不是每个项目都要很大，有些只是给自己留一盏灯。」

## 生成提示词总设定

这是我给丹祈固定下来的核心 prompt：

```text
original virtual avatar for the owner of danzaii.cn, androgynous gentle anime character, medium-long silver white hair with soft pink gradient tips, slightly wavy hair, soft pink and cyan gradient eyes, tiny sheep horn hair accessories, warm tech creator aura, cherry blossom petals, small camera strap, subtle terminal code motifs, pixel bead charm, three tiny cat silhouettes, luminous digital paintbrush, elegant soft pink white and lavender color palette, clean detailed character design, high quality anime illustration, gentle lighting, refined lineart, no real person, no copyrighted character
```

中文拆解一下就是：

- 中长发，银白到浅粉渐变；
- 眼睛是樱粉和湖蓝；
- 有小羊角发饰，但不要太兽化；
- 带相机、终端窗口、像素拼豆、发光画笔；
- 身边可以出现三只猫的剪影；
- 整体是粉白、浅紫、湖蓝的柔和色系；
- 气质是温柔的技术创作者，不是战斗角色。

## 最后

这次做丹祈，其实不是为了给自己强行套一个「人设」。更像是把 danzaii.cn 这几年长出来的东西整理成一个可以被看见的形象。

博客是记忆，作品集是行动，照片墙是现实，AI-Image-Studio 是想象，PixelBead 是手作，Tianji 是一点神秘感，Inside-ME 是向内看的自己。

而丹祈站在这些东西中间，像一只小羊工坊主，手里拿着发光画笔，身边绕着猫，背后是代码、照片、樱花和还没完成的下一个项目。

> _如果一个网站会慢慢长出灵魂，那它大概会先长出一个会写代码、会拍照、也会悄悄收集花瓣的人。_
