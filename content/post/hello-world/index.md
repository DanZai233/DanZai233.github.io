这是我的第一篇博客文章。这个博客使用 [Hugo](https://gohugo.io/) 构建，采用了美观的 [Stack](https://github.com/CaiJimmy/hugo-theme-stack) 主题，并通过 GitHub Actions 自动部署到 GitHub Pages。

## 为什么要写博客？

写博客是一种很好的学习方式。通过将所学知识整理成文字，不仅能加深自己的理解，还能帮助到其他人。

## 技术栈

本博客使用了以下技术：

- **Hugo** — 快速的静态网站生成器
- **Stack 主题** — 卡片式设计，简洁优雅
- **GitHub Actions** — 自动构建和部署
- **GitHub Pages** — 免费的静态网站托管
