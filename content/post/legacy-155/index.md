## 快乐小学期

**小组选题：公路卡口监控系统**

实现卡车车辆的监控，主要包括：
1. 车辆捕获
2. 速度测定
3. 图像记录
4. 车辆识别
5. 信息上传

**感觉难度挺大的**，这当毕设完全可以了我感觉，先慢慢搞吧。

技术选型要求：
- 前后端分离，后端 RESTful API
- 后端 Spring Boot
- ORM 框架 Mybatis/MybatisPlus
- Git 代码托管
- 权限管理框架 Spring Security 或 Shiro
